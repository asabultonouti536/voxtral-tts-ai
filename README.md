# Voxtral TTS AI

A practical repository for exploring [Voxtral TTS](https://voxtral-tts.com) in real text-to-speech workflows.

## Why this repository exists

This project is meant to be a useful public starting point for people who want to understand where [Voxtral TTS](https://voxtral-tts.com) fits in a real workflow. It focuses on practical references and evaluation paths instead of hype.

## Reference implementation

See [mudler/voxtral-tts.c](https://github.com/mudler/voxtral-tts.c) for a concrete implementation reference.

## Practical use cases

Fast TTS evaluation, voice workflow prototyping, creative iteration, and lightweight implementation research.
